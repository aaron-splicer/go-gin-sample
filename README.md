# go-gin-sample

Sample Go + Gin app used by Splicer's real-Railway e2e test rotation (#1041).

- `GET /` — hello-world
- `GET /healthz` — liveness

**Do not delete.** Wired to `ClaudeApiRealRailwayIT.GO_GIN_STACK`.

## Local

```bash
go mod tidy   # populates go.sum on first checkout
go run .
```

## Docker

```bash
docker build -t go-gin-sample .
docker run --rm -p 8080:8080 go-gin-sample
```

## go.sum

`go.sum` is intentionally not committed — it's generated by `go mod download` during the
Docker build (or `go mod tidy` locally). Keeps this skeleton minimal; the build is
reproducible from `go.mod` + the Go proxy.
